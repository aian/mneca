# mneca

The tool-set for making flash cards 


## Overview

The **mneca** is a tools for making flash cards. A flash card is a handy-sized card, which is intended to be used to memorize information. The both side of the card is printed related information (typically `question-and-answer').

For now, the purpose of this project is to create Japanese kanji cards semi-automatically. We are intended to organize it to be able to procecss more generic information in the future.
